### 📌 대표 프로젝트 / Featured Project

#### 📔 Lingu Diary — AI 영어 일기 앱 (iOS)

> AI 피드백을 받으며 매일 영어 일기를 써서 영어 실력을 키우는 모바일 서비스.
> <br/>A mobile app for improving English by writing daily journals with AI-powered feedback.

<p align="center">
  <a href="https://lingudiary.cheeselab.dev">
    <img src="https://lingudiary.cheeselab.dev/og-image.png" width="100%" alt="Lingu Diary"/>
  </a>
</p>

기획 · 개발 · 배포 · 운영까지 제품의 전 과정을 담당했습니다. 모바일 앱, 백엔드, 인앱 결제, 운영 지표 대시보드까지 하나의 제품으로 구축했습니다.
<br/>Owned the product end-to-end — mobile app, backend, in-app payments, and an internal analytics dashboard.

| 영역 / Area | 구현 내용 / What I built |
| --- | --- |
| 📱 **모바일 앱** | React + TypeScript + Vite, **Capacitor**로 iOS/Android 네이티브 빌드 |
| 🤖 **AI** | **Open AI** 연동 — 영어 첨삭, 피드백 팁, 주간/월간 오답(Top3) 리포트 |
| 🗄️ **백엔드** | **Supabase** (PostgreSQL · Auth · Edge Functions), KST 기준 사용량 쿼터 |
| 💳 **결제** | iOS **StoreKit** 자동 구독(월/연), Edge Function 영수증 검증, Free/Plus 티어 |
| 📊 **운영 대시보드** | 별도 **Next.js 16 + React 19** 앱 — DAU/WAU/MAU, MRR/ARR, 전환율, AI 비용 지표 |
